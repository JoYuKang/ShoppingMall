### 🎮 기능 구현 🎮
 #### 공통 기능
  - 회원가입
  - 로그인
  - 등록 상품 확인
  - 상품 판매 상태 확인
  - 카카오 API 사용한 로그인 기능
  - 정보 수정
  #### 관리자 기능
   - 상품 등록
   - 상품 수량 등록
   - 상품 수정
   - 상품 삭제
   - 이미지 등록
  #### 유저
   - 상품 구매
   - 상품 장바구니 넣기
   - 장바구니 상품 구매
   - 구매 기록 확인
   - 상품 구메 취소

### 추가 구현 예상안
